ZIO Redis is a type-safe, performant, ZIO native Redis client.

## Installation

Include ZIO Redis in your project by adding the following to your build.sbt file:

```scala
libraryDependencies += "dev.zio" %% "zio-redis" % "@VERSION@"
```
